## 📝 Style & Principes

### **5 Piliers du Style**

| Pilier | Description | Exemple |
|--------|-------------|---------|
| **Analytique** | Basé sur données statistiques | "moyenne combinée de **0.75 buts**" |
| **Clair** | Facile à comprendre | "Les deux équipes affichent..." |
| **Professionnel** | Vocabulaire technique approprié | "variance combinée", "coefficient de variation" |
| **Prudent** | Notes de mise en garde | "**Note** : L'échantillon de 9 matchs est modéré" |
| **Orienté données** | Chiffres précis | "**55%** de 0-0 HT", "écart de **32.0%**" |

## 📊 Structure des Explications

### **5 Sections Standard**

```
1. OPENING
   → Contexte du match
   → "**Analyse statistique : Team A vs Team B**"

2. CORE STATISTICS
   → Données clés (moyennes, fréquences)
   → "moyenne combinée de **X buts**"

3. HISTORICAL ANALYSIS
   → Analyse historique
   → "Sur les X matchs analysés..."

4. MARKET DISCREPANCY
   → Écart marché-modèle
   → "**Écart de marché** : X%"

5. CONFIDENCE STATEMENT
   → Niveau de confiance + justification
   → "**Confiance** : HIGH (X%) - raisons..."
```

## 🎯 6 Types de Templates

### **1. HT Under/0-0 HT**

**HIGH Confidence** :
```
**Analyse statistique : Wrexham AFC vs Notts County**

Les deux équipes affichent une moyenne combinée de **0.75 buts** en première 
mi-temps sur leurs 12 derniers matchs. Wrexham termine à 0-0 à la mi-temps dans 
**55%** de ses matchs à domicile, tandis que Notts County enregistre **60%** de 
0-0 HT à l'extérieur.

Sur l'ensemble des 24 rencontres analysées, **aucun match n'a dépassé** la ligne 
proposée par le bookmaker.

La variance combinée de **0.45** indique des scores très prévisibles en première 
mi-temps. Les deux formations sont identifiées comme **démarreurs lents** 
(VERY_SLOW et VERY_SLOW), renforçant la probabilité d'un début de match prudent.

**Écart de marché** : Le bookmaker évalue la probabilité à **40.0%**, tandis que 
notre modèle statistique calcule **72.0%**, soit un écart de **32.0%**.

**Niveau de confiance** : HIGH (82%) - La cohérence des données historiques 
(12 matchs) et la stabilité des performances (CV moyen : 0.58) supportent cette 
analyse.
```

**MEDIUM Confidence** :
```
**Analyse Wrexham AFC vs Notts County**

Les statistiques de première mi-temps montrent une moyenne combinée de **0.85 buts** 
sur les derniers matchs. Wrexham enregistre **50%** de 0-0 HT à domicile, Notts 
County **55%** à l'extérieur.

Le modèle identifie un écart de **18.0%** entre l'évaluation du marché (48.0%) et 
les probabilités calculées (66.0%).

**Note** : L'échantillon de 9 matchs est modéré. Une validation supplémentaire est 
recommandée.

**Confiance** : MEDIUM (62%)
```

**LOW Confidence** :
```
**Analyse préliminaire : Team A vs Team B**

Les données disponibles (6 matchs) suggèrent une moyenne de 2.1 buts en première 
mi-temps. Un écart de 12.0% est observé entre le marché et le modèle.

**Avertissement** : L'échantillon de 6 matchs est limité. La stabilité des 
performances est faible (0.45). La variance élevée (0.95) indique une forte 
imprévisibilité. Cette analyse doit être considérée comme **indicative** uniquement.

**Confiance** : LOW (38%)
```

notre modèle (80.0%) suggère une sous-évaluation du potentiel offensif.
