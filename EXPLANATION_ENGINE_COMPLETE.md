**ExplanationEngine complet** pour transformer scores et statistiques en explications lisibles, analytiques et professionnelles.

---

## 📝 STYLE D'EXPLICATION

### **5 Piliers**

| Pilier | Description | Exemple |
|--------|-------------|---------|
| **Analytique** | Basé sur données chiffrées | "Écart de **27.0%** entre bookmaker et modèle" |
| **Clair** | Facile à comprendre | "Le bookmaker sous-évalue cette probabilité" |
| **Professionnel** | Vocabulaire technique approprié | "variance combinée", "coefficient de variation" |
| **Prudent** | Mise en garde explicite | "⚠️ ATTENTION : Confiance faible" |
| **Basé données** | Chiffres précis | "Sur **15 matchs**, **72%** sont 0-0 HT" |

---

## 📊 SECTIONS GÉNÉRÉES

### **7 Sections Automatiques**

```
1. Title
   → Nom du marché analysé

2. Summary
   → Résumé exécutif (écart, confiance)

3. Statistical Analysis
   → Analyse détaillée par marché
   → Variance et stabilité

4. Confidence Explanation
   → Pourquoi HIGH/MEDIUM/LOW
   → Facteurs de confiance

5. Risk Assessment
   → Facteurs de risque identifiés
   → Niveau de risque global

6. Recommendation
   → Recommandation finale
   → Action suggérée

7. Full Text
   → Toutes sections combinées
```

## 🎯 TEMPLATES PAR CONFIANCE

### **1. HIGH Confidence**

**Caractéristiques** :
- ✅ Écart important (>20%)
- ✅ Variance faible (<0.6)
- ✅ Stabilité élevée (>0.70)
- ✅ Échantillon robuste (≥12)
- ✅ Signaux forts multiples

**Exemple** :
```
✅ Anomalie forte (score 85/100) avec confiance élevée. 
Cette opportunité présente un profil risque/rendement favorable 
basé sur l'analyse statistique.

Facteurs de confiance élevée :
1. Écart important de 27.0% entre bookmaker et modèle
2. Variance très faible (82/100) - scores prévisibles
3. Stabilité élevée (85/100) - performances cohérentes
4. Échantillon robuste de 15 matchs
5. 3 signal(aux) fort(s) détecté(s)
```

---

### **2. MEDIUM Confidence**

**Caractéristiques** :
- ⚠️ Échantillon modéré (8-12)
- ⚠️ Variance moyenne (0.6-1.2)
- ⚠️ Écart modéré (15-20%)

**Exemple** :
```
⚠️ Approche prudente recommandée. L'anomalie est détectée 
(score 68/100) mais certains facteurs limitent la confiance.

Facteurs limitant la confiance :
1. Échantillon modéré (10 matchs) - validation supplémentaire recommandée
2. Variance moyenne (58/100) - certaine imprévisibilité
3. Écart modéré (18.5%) - anomalie moins prononcée

Recommandation : Considérer comme opportunité potentielle 
nécessitant validation.
```

---

### **3. LOW Confidence**

**Caractéristiques** :
- ❌ Échantillon petit (<8)
- ❌ Variance élevée (>1.5)
- ❌ Stabilité faible (<0.50)
- ❌ Qualité données faible (<0.7)

**Exemple** :
```
⚠️ ATTENTION : Confiance faible

Facteurs de risque importants :
1. Échantillon très petit (6 matchs) - fiabilité limitée
2. Variance élevée (35/100) - scores imprévisibles
3. Stabilité faible (42/100) - performances incohérentes

⚠️ AVERTISSEMENT : Cette analyse doit être considérée comme 
purement indicative. Le risque de faux positif est élevé. 
Ne pas utiliser comme base de décision sans validation externe.
```

## 🎯 TEMPLATES PAR MARCHÉ

### **1. HT Markets (HT Under/Over)**

**Focus** :
- Taux 0-0 HT historique
- Variance première mi-temps
- Profil démarreurs lents/rapides

**Exemple** :
```
Analyse Première Mi-Temps

Le taux de réussite historique pour ce marché est de 72.5% 
sur les 15 derniers matchs analysés. La variance des scores 
en première mi-temps est très faible (82/100), indiquant des 
performances hautement prévisibles. Les équipes montrent un 
profil de démarreurs lents confirmé.
```

---

### **2. Extreme Under (6.5, 8.5, 10.5)**

**Focus** :
- Distance statistique de la ligne
- Taux dépassement historique
- Cohérence variance

**Exemple** :
```
Analyse Ligne Extrême

La ligne proposée (10.5) est statistiquement très éloignée 
des performances observées. Sur les 15 matchs analysés, 95% 
sont restés sous cette ligne, suggérant une surévaluation 
significative par le bookmaker. La variance faible (78/100) 
renforce la cohérence de ces statistiques.
```

---

### **3. BTTS (Both Teams To Score)**

**Focus** :
- Taux BTTS historique
- Capacités offensives
- Écart évaluation

**Exemple** :
```
Analyse Both Teams To Score

Le taux historique de matchs avec les deux équipes marquant 
est de 68.5%. L'écart de 22.0% entre le bookmaker et le modèle 
suggère une évaluation incorrecte des capacités offensives.
```

---

### **4. Standard FT Markets**

**Focus** :
- Taux réussite historique
- Discrepancy score
- Variance et stabilité

**Exemple** :
```
Analyse Match Complet

Les statistiques historiques montrent un taux de réussite de 
65.0% pour ce marché. Un écart significatif (68/100) est observé.
```

## 💻 UTILISATION

### **Basique**

```python
from app.services.explanation import ExplanationEngine

# Create engine
explanation_engine = ExplanationEngine()

# Generate explanation
explanation = explanation_engine.generate_explanation(anomaly_result)

# Access sections
print(explanation['title'])
print(explanation['summary'])
print(explanation['full_text'])
```
